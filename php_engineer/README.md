## Desafio PHP Engineer

Crie uma API Restful que implemente o CRUD de filmes e categoria de filmes.

Categoria | Tipos |
--- | --- |
id | PK |
name | string |
                                
Filme | Tipos |
--- | --- |
id | PK |
category | FK |
name | string |
directed_by | string |
year | integer |

## Requisitos
- Banco de dados: MongoDB, MySQL ou PostgreSQL
- Testes unitários e testes de integração
- Conhecimentos de Restful
    - HTTP Protocol
    - Status Code
    - Formato de URL's
